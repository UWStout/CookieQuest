Installation: 1.Download repository
   If installing for Web Use, put the HTML_Build on a web server and access the game thru the IP address. 
   If installing for Local Machine, go into the PC Build folder, and run the executable file, game should launch as an application. 

Description of Game: Cookie Quest is a 2d boss battler, take on the quest of a dino baker to collect his cookie ingredients from all
around the world. Collect these ingredients by battleing the sentient ingedient based monsters and collect a piece of them.

Keyboard Mappings:
  Right and Left Arrows: Movement left and right
  Up Arrow: Jump
  Space: Hit/Swing Pin
  Esc: Pause/Unpause
